# デュエル大富豪 (Duel Daifugo)

[**ブラウザでプレイ**](https://tomotomo.github.io/vibe-game/) | [English](./README.md)

Unity WebGLで作られた、シンプルな1対1の大富豪ゲームです。

## GitHub Pages

以下のURLからブラウザで直接プレイできます：
https://tomotomo.github.io/vibe-game/

## 特徴

- **1対1 対戦**: 戦略的なCPU対戦相手とプレイします。
- **クラシック & ローカルルール**:
  - **8切り**: 8を出すと場が流され、もう一度自分のターンになります。
  - **5スキップ**: 5を出すと対戦相手のターンをスキップします。注意：場は**流れない**ため、自分で出した5よりも強いカードを続けて出す必要があります。
  - **11バック (Jバック)**: ジャック(11)を出すと、一時的にカードの強さが反転します。すでに革命状態の場合は**カウンター革命**として機能し、強さが通常に戻ります。
  - **縛り**: 直前のカードと同じスート（マーク）を出すと、そのターンの間スートが固定されます。複数枚出しの場合は、スートの組み合わせが完全に一致する必要があります。
  - **革命**: 同じ数字のカードを4枚以上出すと、カードの強さが反転します。
- **戦績データ保存**: 総試合数、勝利数、現在の連勝数、最大連勝数をローカルストレージに保存・記録します。
- **ダイナミックな開始**: ゲーム開始時にサイコロを振るアニメーションで初期手札の枚数が決まります。
- **スマートUI**: 
  - **親のターン (Lead)**: 複数のカードを選択して「PLAY」ボタンを押します。
  - **子のターン (Follow)**: カードをクリックすると、有効な組み合わせを自動判定して即座に出します（スマートプレイ）。スマートプレイは、縛り発生時の条件も考慮して賢く判定します。
- **リプレイ性**: 終了後、すぐにゲームを再開できます。

## 技術スタック

- **エンジン**: Unity 2022.3
- **言語**: C#
- **UI**: Unity UI (uGUI) - スクリプトによる動的なレイアウト生成。
- **開発ツール**: [Gemini CLI](https://github.com/google/gemini-cli) - すべてのコード、ロジック、アセットは自然言語のプロンプトを通じて生成されました。

## ビルド手順

1. **前提条件**: Unityがインストールされていることを確認してください（2022.3 LTS推奨）。
2. **クローン**: このリポジトリをクローンします。
3. **開く**: Unity Hubでプロジェクトフォルダを開きます。
4. **シーン**: `Assets/Scenes/MainScene.unity`を開いてください。
5. **プレイ**: Unityエディタの再生ボタンを押します。
6. **ビルド**: `File > Build Settings` に移動し、プラットフォーム（PC, Mac, WebGLなど）を選択して `Build` をクリックします。

## Gemini CLI による開発

このプロジェクトは、AIエージェントを使用した「バイブコーディング (Vibe Coding)」のデモンストレーションとして機能します。

- **役割**: AIが唯一の開発者として機能し、高レベルの指示に基づいてスクリプトの記述、エラーのデバッグ、機能の実装を行いました。
- **ワークフロー**:
    1. ユーザーが機能をリクエスト（例：「大富豪ゲームを作って」）。
    2. エージェントが初期のC#スクリプトを生成。
    3. ユーザーがリクエストを洗練（例：「手札枚数をサイコロで決めるようにして」「UI操作を修正して」）。
    4. エージェントがコードを修正し、Gitに変更をコミット。
