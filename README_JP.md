# 大富豪 (Daifugo) - 1対1 カードゲーム

[English](README.md) | [日本語](README_JP.md)

Unityで構築され、Gemini CLIを使用して生成された、日本の人気カードゲーム「大富豪」のシンプルかつ魅力的な1対1実装です。

## 特徴

- **1対1 対戦**: CPU対戦相手とプレイします。
- **クラシックルール**: 以下のような標準的なルールを含みます：
  - **8切り**: 8を出すと場が流され、もう一度自分のターンになります。
  - **5スキップ**: 5を出すと対戦相手のターンをスキップします。
  - **11バック (Jバック)**: ジャック(11)を出すと、一時的にカードの強さが反転します（革命状態）。
  - **縛り**: 直前のカードと同じスート（マーク）を出すと、そのターンの間スートが固定されます。
- **ダイナミックな開始**: ゲーム開始時にサイコロを振るアニメーションで初期手札の枚数が決まります。
- **スマートUI**: 
  - **親のターン (Lead)**: 複数のカードを選択して「PLAY」ボタンを押します。
  - **子のターン (Follow)**: カードをクリックすると、有効な組み合わせを自動判定して即座に出します（スマートプレイ）。
- **リプレイ性**: 終了後、すぐにゲームを再開できます。

## 技術スタック

- **エンジン**: Unity 2022.3 (または互換バージョン)
- **言語**: C#
- **UI**: Unity UI (uGUI) - スクリプトによる動的なレイアウト生成（プレハブ不要）。
- **開発ツール**: [Gemini CLI](https://github.com/google/gemini-cli) - すべてのコード、ロジック、アセットは自然言語のプロンプトを通じて生成されました。

## ビルド手順

1. **前提条件**: Unityがインストールされていることを確認してください（2021.3 LTS以降推奨）。
2. **クローン**: このリポジトリをクローンします。
3. **開く**: Unity Hubでプロジェクトフォルダを開きます。
4. **シーン**: ゲームロジックは自動的に起動します。`Assets/Scenes/SampleScene.unity`を開くか、新しい空のシーンを作成してください。
5. **プレイ**: Unityエディタの再生ボタンを押します。
6. **ビルド**: `File > Build Settings` に移動し、プラットフォーム（PC, Mac, WebGLなど）を選択して `Build` をクリックします。

## Gemini CLI による開発

このプロジェクトは、AIエージェントを使用した「バイブコーディング (Vibe Coding)」のデモンストレーションとして機能します。

- **役割**: AIが唯一の開発者として機能し、高レベルの指示に基づいてスクリプトの記述、エラーのデバッグ、機能の実装を行いました。
- **ワークフロー**:
    1. ユーザーが機能をリクエスト（例：「大富豪ゲームを作って」）。
    2. エージェントが初期のC#スクリプトを生成。
    3. ユーザーがリクエストを洗練（例：「手札枚数をサイコロで決めるようにして」「UI操作を修正して」）。
    4. エージェントがコードを修正し、Gitに変更をコミット。
